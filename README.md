Objective
To design and simulate a 4-bit counter using Verilog. This repository contains the HDL code, functional simulation, and waveform outputs demonstrating correct operation.

Files Included
counter_4bit.v: The main Verilog design code.
tb_counter_4bit.v: The testbench code to simulate the design.
Waveform Screenshot: Visual proof of the counter incrementing correctly from 0 to 15.

Tools Used
Simulation: EDA Playground
Simulator: Icarus Verilog
Viewer: EPWave
